O que fazemos
Coleta de Dados

Os dados são coletados diretamente das APIs. Cada uma delas fornece dados específicos relacionados a vídeos, tendências de pesquisa e interações em subreddits.

Análise Estatística

São calculadas estatísticas descritivas para entender a distribuição dos dados e identificar padrões. Isso inclui o cálculo de médias e a identificação de outliers.

Modelagem dos Dados

Os dados coletados são organizados em modelos específicos para cada categoria, contendo métricas detalhadas, facilitando a análise e identificação de tendências.
